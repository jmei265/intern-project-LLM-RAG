Network logon scripts can be assigned using Active Directory or Group Policy Objects.[1]These logon scripts run with the privileges of the user they are assigned to. Depending on the systems within the network, initializing one of these scripts could apply to more than one or potentially all systems.Adversaries may use these scripts to maintain persistence on a network.
